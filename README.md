## 🧠 Sobre o projeto

🦇 O **Wayne Finance Dashboard** é uma aplicação moderna para gerenciamento de despesas pessoais.

### 🎯 Objetivo

- Criar um dashboard financeiro moderno
- Praticar integração entre front-end e back-end
- Aplicar boas práticas com React + TypeScript

---

## 🚀 Funcionalidades

- 📝 Cadastro de despesas  
- 📋 Listagem em tempo real  
- ❌ Exclusão de registros  
- 🔍 Filtro por busca  
- 📊 Resumo financeiro automático  
- 📱 Interface responsiva  
- 🌙 Tema dark premium  

---


## 🛠️ Tecnologias

### 💻 Front-end
- ⚛️ React
- 📘 TypeScript
- 🎨 Tailwind CSS
- ⚡ Vite

### 🔙 Back-end
- ☕ Spring Boot
- 🗄️ JPA / Hibernate
- 💾 Banco H2
